# BoostPC

Windows utility for safe system optimization with real-time CPU, GPU, and RAM monitoring, animated circular graphs, and memory/system boost features.

## Features
- Real-time CPU, GPU, RAM monitoring
- Circular graphs with smooth animation
- System Boost (Safe CPU & system optimization)
- Memory Cleaner (Free RAM, remove temp files)
- Multi-language support: Thai / English / Chinese
- Admin detection and display
- Interactive dashboard for system management
